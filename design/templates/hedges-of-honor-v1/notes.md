# Hedges of Honor

Reference-first black-border ScienceCard variant with the same card proportions as the canonical ScienceCard shell.

## 2026-03-20

- Added a new template variant for the white-card maritime reference.
- Kept the same science-card geometry so the card ratio stays stable.
- Used the ScienceCard base proportions instead of the SkyFrame variant so the card reads like a card, not a frame.
- Switched the card to a sharp rectangular silhouette so Hedges stays hard-edged instead of inheriting ScienceCard rounding.
- Switched the border to a 2px black stroke and kept the subtle inset shadow for volume.
- Reused the sky backdrop pattern so the background actually applies in runtime instead of falling back to a blank canvas.
- Remaining work: if a future reference image lands, calibrate the backdrop cloud shapes and card shadow more tightly.
